# アイコン画像の作成手順

PWA対応には以下のアイコン画像が必要です：

## 必要なファイル
- `public/icon-192.png` (192x192ピクセル)
- `public/icon-512.png` (512x512ピクセル)

## 作成方法

### オンラインツールを使う場合
1. [Canva](https://www.canva.com/) や [Figma](https://www.figma.com/) で正方形のデザインを作成
2. けん玉の絵文字 🪀 や文字「け」などをデザイン
3. 背景色は緑系（#10b981）を推奨
4. 192x192と512x512でエクスポート

### AIツールを使う場合
1. DALL-E や Midjourney で「kendama icon, simple, flat design, green theme」と入力
2. 正方形で出力
3. 必要なサイズにリサイズ

### シンプルな方法
1. 適当な画像エディタで正方形の画像を作成
2. 背景を緑（#10b981）に
3. 中央に「けん玉」か「🪀」を配置
4. 192x192と512x512で保存

## 配置
作成した画像を以下に配置してください：
```
kendama-app/
  public/
    icon-192.png
    icon-512.png
```

配置後、ブラウザで開いて「ホーム画面に追加」を試してください。
