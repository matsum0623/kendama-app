# 🪀 けん玉練習アプリ - MVP

日本けん玉協会の練習を継続しやすくする、スマホ最適化WEBアプリです。

## ✅ 実装済み機能（MVP）

### 基本機能
- ✅ ホーム画面の実装
- ✅ 成功/失敗ボタン
- ✅ 連続成功数（streak）のカウント
- ✅ 累計成功数のカウント
- ✅ localStorage による永続化（リロード後も保持）

### UI/UX
- ✅ スマホファースト設計
- ✅ 片手・親指操作に最適化
- ✅ タッチフレンドリーなボタンサイズ
- ✅ レスポンシブデザイン
- ✅ 横向き対応
- ✅ PWA対応メタタグ

### データ管理
- ✅ クライアントサイドでlocalStorageを使用
- ✅ ハイドレーションエラー回避
- ✅ 日本語ロケール対応

## 🚀 開発方法

### 起動
```bash
cd d:\code\kendama-app
npm run dev
```

http://localhost:5173/ にアクセス

### ビルド
```bash
npm run build
npm run start
```

## 📱 使い方

1. アプリを開く
2. 「大皿」の練習を行う
3. 成功したら「✓ 成功」ボタンをタップ
4. 失敗したら「○ 失敗」ボタンをタップ
5. 連続成功数と累計成功数が自動で記録される

### 挙動
- **成功**: 連続成功 +1、累計成功 +1
- **失敗**: 連続成功リセット（累計は減らない）
- データは自動保存（localStorage）

## 📝 注意事項

### 現在の制限事項
- ログイン機能なし（固定で "guest" ユーザー）
- 技は「大皿」固定
- 日付別管理なし
- データのクラウド同期なし
- 級・段の判定なし

### ブラウザサポート
- モダンブラウザ（Chrome, Safari, Edge最新版）
- localStorage対応が必須

## 🔐 データの扱い

- すべてのデータはブラウザのlocalStorageに保存
- サーバーには一切送信されない
- ブラウザのキャッシュクリアで消去される
- 将来的にAWS DynamoDBに移行予定

## 🐛 トラブルシューティング

### データが保存されない
- ブラウザのプライベートモードでないか確認
- localStorageが有効か確認
- ブラウザコンソールでエラーを確認

### レイアウトが崩れる
- キャッシュをクリアして再読み込み
- 開発サーバーを再起動
